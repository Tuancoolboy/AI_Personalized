# Frontend — Next.js 16

Web app AI Trợ Lý (App Router). Next.js project root khi chạy `npm run dev` / `npm run build` từ repo root.

```
src/frontend/
├── app/          # Routes, layouts, API route handlers
├── components/
├── hooks/
├── lib/
├── public/
└── proxy.ts      # Auth/session guard (Next.js 16 proxy)
```

Import alias: `@/*` → thư mục này.
